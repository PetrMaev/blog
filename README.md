# Проект приложения платформы ведения блога
## Описание:
**Приложение для публикации личных постов.**
## Цель проекта:
Реализовать приложение, которое позволяет пользователям публиковать посты с возможностью оставлять комментарии
## Установка:
1. Клонируйте репозиторий:
```python
https://github.com/PetrMaev/blog.git
```
```python
git@github.com:PetrMaev/blog.git
```
2. Установите зависимости:
```python
poetry install
```
## Наполнение данными базы данных:
Для наполнения данными базы данных используйте фикстуры и кастомные команды.
1. Кастомная команда для заполнения базы данных тестовыми данными:
```python
python manage.py add_data
```
2. Кастомная команда для создания суперпользователя:
```python
python manage.py csu
```
3. Кастомная команда для заполнения базы данных пользователями для тестирования:
```python
python manage.py add_users
```
## Использование:
Для запуска приложения введите в терминале следующую команду:
```python
python manage.py runserver
```
## В проекте реализованы:
1. Регистрация пользователей.
2. Авторизация.
3. Вывод в админ-панели созданных объектов.
4. Фильтрация в админ-панели по дате создания поста.
5. Валидатор для пароля - пароль должен быть не менее 8 символов, должен включать цифры.
6. Валидатор для почты. Разрешенные домены: mail.ru, yandex.ru.
7. Проверка того, что автор поста достиг возраста 18 лет.
8. Проверка, что автор в заголовок не вписал запрещенные слова.
